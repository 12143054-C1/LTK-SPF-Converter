Handling Conversion Failures

If the conversion process fails, a popup window titled "Problem Detected" will appear. This window offers three options: Send Log, Help, and Cancel.

  • Send Log: Clicking this button will open a new email composition window in Outlook with the log file and other relevant files already attached for debugging. You have the option to add a personal message describing the issue in detail, which can help the developers diagnose the problem more efficiently. However, adding a personal message is optional.
  • Help: Pressing the Help button will direct you to this page for more detailed instructions on how to handle conversion failures.
  • Cancel: Closes the popup window and cancels the error reporting process.

The support team will review the log and respond with a resolution as soon as possible.
