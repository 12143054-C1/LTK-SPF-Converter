LTK SPF Converter Help

Welcome to the help documentation for the LTK SPF Converter application.

This program is designed to convert .spf files used in the HVM (High Volume Manufacturing) environment to Python files that are compatible with the PythonSV environment.

Usage

For detailed instructions on using the application, please refer to the corresponding sections below.

  • Handling Conversion Faliures

Workflow

Follow these steps to use the LTK SPF Converter:

  1. Open the application by running the Python script.
  2. In the "Source" section, select whether you want to convert a file or a folder by choosing the appropriate radio button.
  3. Click on the "Select File / Folder" button to open a file dialog and choose the file or folder you want to convert.
  4. Once selected, the path of the file or folder will appear in the combo box next to the button.
  5. In the "Output" section, click on the "Select Output Folder" button to choose where you want the converted files to be saved.
  6. Alternatively, you can click on the "Default Output Folder" button to set a default folder for the output.
  7. Specify any additional conversion options in the "Options" section:
    • Choose the CPU generation from the "CPU Gen" dropdown menu.
    • Check the "Use itpp Comments" checkbox if you want to include itpp comments in the conversion.
  8. Once all options are set, click on the "Convert" button in the bottom right corner to start the conversion process.
  9. A message box will appear showing the details of the conversion, including the source and destination paths, CPU generation, and whether itpp comments are used.

Tips

  • The destination folder is set automatically when the source file or folder is selected.
  • Make sure to check the conversion details in the message box before proceeding.
  • You can clear the history of selected files and folders from the top menu.
