Selecting Output Folder

To select the output folder where converted files will be saved, follow these steps:

  1. In the "Destination" section, click the "Select Output Folder" button to open a folder dialog.
  2. Navigate to the desired output folder and select it.
  3. The path of the selected output folder will appear in the combo box next to the button.

Alternatively, you can click the "Default Output Folder" button to set a default folder for the output, which will be created based on the source folder's location and the current timestamp.
