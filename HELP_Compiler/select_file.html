Selecting Input File/Folder

To select the input file or folder, follow these steps:

  1. Open the application and look at the "Source" section.
  2. Select either the "File" or "Folder" radio button based on what you want to convert.
  3. Click the "Select File / Folder" button to open a file dialog.
  4. Navigate to the desired file or folder and select it.
  5. The path of the selected file or folder will appear in the combo box next to the button.
