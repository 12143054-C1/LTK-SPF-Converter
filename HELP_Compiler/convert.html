Converting Files

To convert files, follow these steps:

  1. Ensure that you have selected both the input source and output destination.
  2. Check the conversion options and make any necessary changes.
  3. Click the "Convert" button located in the bottom right corner of the application.
  4. A message box will appear showing the details of the conversion. Verify these details before proceeding.
  5. After verifying the details, the conversion process will start and the converted files will be saved to the selected output folder.
